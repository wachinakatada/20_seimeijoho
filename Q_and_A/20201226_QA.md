| 授業で使用しているさまざまなウェブツールは、一回で信頼性が100%と言えるのか | ツールを手放しで信頼していいのか、という質問かと思ったのですが、その意味では、それぞれのツールの元論文を読み込むなどして、正しい使い方や限界を理解をした上で使用しないと、全く間違ったことをしてしまう危険があると思います。 |
| 今回，練習で作ったコンティグのどこがだめで，どう改善すべきなのかが分からなかったです | 理想的にはコンティグの数は、細菌なら染色体とプラスミドの数になるはずです。練習で行ったように短いリードのみでアセンブルをするとうまくつなげることが出来ず、多くのコンティグが出来てしまいます。さらにそのコンティグをつなげるには、メイトペアという手法を用いたり、最近ではロングリードシーケンサー（数kbのリードが得られる）を用いたりして得たリードを合わせてアセンブルする必要があります。 |
| Raw  dataのqualityの一覧にあるQとPeの意味を教えてください。  | Qはquality  score、Peはその場所（何塩基目か）におけるエラーの確率です。下記に説明書のリンクを貼っておきます。ぜひ確認してみて下さい。      https://github.com/torognes/vsearch/releases/download/v2.15.1/vsearch_manual.pdf |

| 微生物の進化というのは具体的にはどのような進化の仕方をするのでしょうか？ | 突然変異、遺伝子の水平伝播、ゲノム再編成などのメカニズムが知られています。 |
| 1％しか培養出来ないとありますが、1％しか詳しく研究出来ないということですか？ | 培養できないと詳しく研究できないわけではないと思います。培養が困難な微生物は、培養を介さずに抽出したDNAやRNAを用いて情報を得ます。培養できる微生物は詳しく研究できますが、培養系における微生物の情報は、培養系における微生物の情報であって、自然界におけるその微生物のふるまいと異なる可能性があることに要注意です。 |
| Pseudonocardia属はジウロンなどの分解能を持つという事でしたが、分解能を獲得するメカニズムについて教えて頂きたいです。 | Pseudonocardia属に属する、我々が分離した菌がジウロンの分解能をもつということです。この属の微生物が一般にジウロン分解能をもつわけではありません。分解能を獲得するメカニズムはまさに解明したいテーマで、現段階では何とも言えません。分解酵素遺伝子の変異集積による機能変化もあるでしょうし、水平伝播で分解酵素遺伝子群を獲得することもあると思います。 |
| これから微生物にかかわる学生にはどんな技能や知識が必要だとお考えですか？ | これから研究を始める学生には、興味、熱意、学習意欲と、粘り強さを持っていただきたいと思います。研究を始める段階では、技能・知識は無いよりはあった方が良いと思いますが、それよりもこれから学ぶことをしっかり身につけていって下さい。 |
| その場の状況が刻々と変化したりと，メタゲノムの実験は再現性をとるということが難しいのでは？と感じました。データのとり方としては，実験回数を増やすのか，一回の実験のデータ収集量を増やすのでしょうか。 | 自然環境のデータをとることについてのコメントとしてお答えします。室内実験より再現性はとりづらいです。実験回数を増やすか、１回のデータ取得量を増やすのかはケースバイケースですが、自然界で起こっていることについて何かを言うならば、少なくとも２年繰り返すことが多いです。 |
| ゲノムシークエンスでゲノムを繰り返し読まれた回数を x  20, x 40 と表すといういう理解で良いか？また、x 数が多いほど良い結果が得られるイメージだが、講義のなかで x 400  だとやばい？的なことを仰られていた気がするのですが、どのような意味なのでしょうか？ | （推定）ゲノム長の２０倍の塩基対を読むことをx20で読むといいます。データが多いと、その分、エラーがデータに入りやすくなり、アセンブルがかえってうまくいかないことがでてきます。また、データ量が多いと計算に時間がかかるようになるのも厄介です。 |
